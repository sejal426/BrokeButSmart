# BrokeButSmart

**Financial literacy for Indian students — learn real money skills with zero real risk.**

> Trade stocks, manage credit, pay off debt, and learn investing — all with fake rupees.

##  The Problem

Most Indian students graduate with zero practical money knowledge. No one teaches you about CIBIL scores, SIPs, or how a 36% APR credit card destroys your finances — until it's too late.

##  What We Built

A browser-based financial simulation game built specifically for India. Pick a starting balance, trade real NSE/BSE-listed stocks, simulate credit events, take loans, and work through bite-sized lessons — all without touching real money.

##  Features

- 📈 **Stock Market Simulator** — trade 15 real Indian stocks (TCS, Reliance, Zomato, etc.) with live-simulated prices
- 💳 **CIBIL Score Engine** — simulate on-time payments, missed EMIs, and high utilisation to see your score move
- 💸 **Debt & EMI Tools** — take loans, practice repayment, built-in EMI calculator
- 🎓 **7 Lessons** — inflation, budgeting, SIPs, taxes, credit cards — all India-specific
- 🎮 **XP & Levels** — gamified progression to keep you coming back

##  Tech Stack

React · Supabase · Vanilla CSS · v0.dev

##  Run Locally
```bash
git clone https://github.com/sejal426/BrokeButSmart.git
cd BrokeButSmart
# Add your Supabase URL and anon key to app.js
open index.html
```

No npm. No build step. Just three files.
